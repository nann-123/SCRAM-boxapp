# Demo Walkthrough

1. 双击启动 GUI。
2. 如需切换语言，在工具栏右侧选择 `zh_CN` 或 `en_US`。
3. 默认模板为 `GMD Greater Paris scenario D (full dynamics)`，用于比较 internal mixing 与 external mixing 假设。
4. 如需换案例，可在模板下拉菜单中选择 `GMD hazy` 验证案例或 Greater Paris 场景 A/B/C/D。
5. 在实验设置页确认混合假设、过程开关、模拟时长、最小时间步和输出目录。
6. 在结构编辑页检查 species、size bins、fraction、emission 和 initial mass 表格。专业用户可直接编辑这些表格后保存 cfg。
7. 点击 `比较 internal / external` 运行标准对比实验。
8. 在运行监控页查看 wall-clock、模拟小时、ETA、总质量、总数浓度和日志。
9. 在结果分析页查看质量/数浓度曲线、相对 external 差异、external mixed fraction 和 mixed/unmixed mass by size。
10. 在报告页勾选图表并生成 PDF 报告。
