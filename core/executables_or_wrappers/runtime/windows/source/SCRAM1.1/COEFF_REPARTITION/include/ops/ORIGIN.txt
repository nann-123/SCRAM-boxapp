Description
-----------
Ops is a C++ library for parsing Lua configuration files. It provides a
programmatic interface to read, write, and manipulate Lua-based configuration,
with SWIG bindings for Python.

Integration into SCRAMBoxApp
-----------------------------
This directory was converted from a git submodule to a regular directory on
2026-07-15 for portability reasons:

  - The original remote (canis01) is a private server, inaccessible to
    third-party developers.
  - Ops has not been updated since 2010 and is treated as a stable, static
    dependency.
  - A regular directory integrates cleanly with PyInstaller packaging and
    CESM build systems.

All source files, copyright notices, and the LGPL 2.1 license are preserved
intact. The SConstruct build script has been adapted for Windows/MSYS2
environments while retaining the original LGPL header.
